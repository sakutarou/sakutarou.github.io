栗栗在木工作室

小小的工作室

大大的可能性

櫻太郎

我是偽前端工程師一枚的櫻太郎，這個栗栗在木工作室是八月底左右開始做的平台。原本是希望能做一個部落格，放點自己的東西，寫點自己的日誌之類的，好不容易終於有暑假可以弄，結果做一做便跳進了前端工程的池子裡，現在朝著鑽研出更好的網站前進中。


For 輕卡：雖然我自己剛開始學沒很久，但還是希望可以透過在貴平台幫忙的經驗，提升自己也幫助你們，基本的 HTML、CSS的操作都沒有問題。倒是 React JS 完全沒用過，現在導覽列那邊是用 Vue 去實作的，所以這部分可能要多擔待。


另外之前在觀察你們的官網時發現你們網頁裡右邊的漢堡選單的地方是跳錯的，要把 parallax-nav 裡面 a 的 href 改到正確的位置(估計只是位子輸入錯而已)。
